# Prototipo Fila zero

>Criei um prototipo de aplicativo que facilita o cliente à sair de filas em restaurantes, permitindo entrar na fila pelo celular.

## Objetivo 
O prototipo tinha que ter:

1. Tela de Login / Cadastro
2. Tela inicial com estabelecimentos próximos
3. Tela de detalhes do estabelecimento
4. Tela para entrar na fila
5. Tela de status da fila / tempo de espera
6. Tela de avaliação do local

> Na **Tela de login**, fiz uma tela simples para o usúario criar sua conta com nome de usúario e senha

> Na **Tela inicial**, fiz uma tela com um ícone de pesquisa, para pesquisar os locais e alguns locais próximos sugeridos para o usuário.

> Na **Tela de detalhes** fiz uma tela que o usuário pode ver um pouco do estabelecimento, dos produtos e entrar na fila do restaurante.

> Na **Tela para entar na fila** fiz uma tela que fala quanto tempo a fila está demorando e da opção para o usuário entrar na fila ou voltar para os estabelecimentos.

> Na **Tela de status** Fiz uma tela que mostra a posição do usuário na fila, quanto tempo demora e um aviso que assim que estiver chegando a vez dele o app manda uma notificação.

> Na **Tela de avaliação** Fiz uma tela simples com 5 estrelas, para o usuário avaliar.
